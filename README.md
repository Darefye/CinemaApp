# О проектe.
Мобильное приложение для поиска фильмов и сериалов, а также управления коллекциями фильмов.  
В роли источника данных выступает [Kinopoisk Api Unofficial](https://kinopoiskapiunofficial.tech/)

## Стек технологий  
MVVM    
Dagger Hilt
Retroffit 2  
Onboarding
Room  
Navigation
Glide 
Moshi  
Custom View
